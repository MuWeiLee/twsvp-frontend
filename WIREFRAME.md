# Wireframe

## Search（搜索）

```
┌──────────────────────────────┐
│ 导航栏（固定）               │
├──────────────────────────────┤
│ 搜索框 + 搜索按钮（固定）    │
├──────────────────────────────┤
│ 内容区（可滚动）             │
│ - 结果 Tabs                  │
│ - 股票/观点/用户列表         │
│ - 空态/加载态                │
├──────────────────────────────┤
│ 免责声明（固定）             │
├──────────────────────────────┤
│ 底部 Tabbar（固定）          │
└──────────────────────────────┘
```

- 搜索框固定在导航栏下方。
- 内容区仅在搜索框与底部 Tabbar 之间滚动。
- 免责声明固定在 Tabbar 上方，不参与滚动。

## StockFeed（个股详情）

```
┌──────────────────────────────┐
│ 导航栏（固定）               │
├──────────────────────────────┤
│ 日K 行情卡片                 │
│ - K 线 + 指标悬浮信息         │
├──────────────────────────────┤
│ 近 7 日观点统计               │
├──────────────────────────────┤
│ 筛选 Tabs + 状态筛选          │
├──────────────────────────────┤
│ 观点列表（可滚动）            │
├──────────────────────────────┤
│ 底部 TabBar（固定）            │
│ - 交易(双向上下箭头) / 发表观点 / 分享(纸飞机) │
└──────────────────────────────┘
```

- TabBar 固定在底部，内容区需预留底部空间避免遮挡。
- 交易图标为双向上下箭头，分享图标为纸飞机（Telegram 风格）。
