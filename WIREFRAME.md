# Wireframe

## 全局设置
- 默认语言：繁体中文（zh-Hant）。

## Search（搜索）

```
┌──────────────────────────────┐
│ 导航栏（固定）               │
│ - 搜索按钮 / 通知按钮         │
├──────────────────────────────┤
│ 搜索框 + 搜索按钮（固定）    │
├──────────────────────────────┤
│ 输入联想下拉选单（插入）     │
├──────────────────────────────┤
│ 内容区（可滚动）             │
│ - 结果 Tabs（股票/资讯/观点/用户） │
│ - 股票/观点/用户列表         │
│ - 空态/加载态                │
├──────────────────────────────┤
│ 免责声明（固定）             │
├──────────────────────────────┤
│ 底部 Tabbar（固定）          │
└──────────────────────────────┘
```

- 搜索框固定在导航栏下方。
- 输入联想下拉选单插入在搜索框与结果 Tabs 之间。
- 输入联想下拉选单最多显示 5 条，多选项时固定为 5 条高度。
- 精准匹配单一选项时，下拉选单高度仅一行。
- 内容区仅在搜索框与底部 Tabbar 之间滚动。
- 免责声明固定在 Tabbar 上方，不参与滚动。
- 页面切换进入时自动回到顶部，不保留上一页滚动位置。

## Feed（观点流）

```
┌──────────────────────────────┐
│ 导航栏（固定）               │
│ - 搜索按钮 / 通知按钮         │
├──────────────────────────────┤
│ 筛选 Tabs（固定）            │
├──────────────────────────────┤
│ 观点列表（可滚动）            │
│ - 中型观点卡片               │
│   - 底部：留言数 icon（点赞左侧，右对齐） │
├──────────────────────────────┤
│ 底部 Tabbar（固定）          │
└──────────────────────────────┘
```

- 下拉刷新仅在观点列表内容区触发。

## News（资讯）

```
┌──────────────────────────────┐
│ 导航栏（固定）               │
│ - 方形 Logo / 标题左对齐 / 右侧搜索按钮 + 通知按钮 │
├──────────────────────────────┤
│ 资讯列表（可滚动）            │
│ - 紧凑卡片：标题/摘要/时间来源 │
│ - 摘要最多 4 行              │
│ - 空态/加载态/下拉刷新        │
├──────────────────────────────┤
│ 免责声明/观点提示语（固定）    │
├──────────────────────────────┤
│ 底部 Tabbar（固定）          │
└──────────────────────────────┘
```

- 资讯列表与导航栏之间保留间距。
- 免责声明固定在 Tabbar 上方，左对齐，不参与滚动。
- 列表滚动到底时，卡片紧贴底部，不留过大空白。
- 页面底色为浅灰，参考观点流/通知中心。
- 资讯卡片无圆角，样式对齐观点卡片。

## Notification（通知）

```
┌──────────────────────────────┐
│ 导航栏（固定）               │
├──────────────────────────────┤
│ 通知 Tabs（固定）             │
│ - 点赞 / 留言 / 到期提醒       │
├──────────────────────────────┤
│ 通知列表（可滚动）            │
│ - 留言卡片                   │
│   - 头像 / 昵称在你的观点下留言 │
│   - 留言内容...               │
│ - 空态/加载态/下拉刷新        │
├──────────────────────────────┤
│ 免责声明（固定）             │
├──────────────────────────────┤
│ 底部 Tabbar（固定）          │
└──────────────────────────────┘
```

- 下拉刷新仅在通知列表内容区触发。
- 列表背景为浅灰，卡片白底无圆角，卡片内容紧凑。
- Tabs 模块与底线样式对齐观点流页面。

## StockFeed（个股详情）

```
┌──────────────────────────────┐
│ 导航栏（固定）               │
├──────────────────────────────┤
│ 日K 行情卡片                 │
│ - 30 根 K 线               │
│ - 浅灰网格线背景             │
│ - 左右轴 + 时间轴 + 点击提示 │
├──────────────────────────────┤
│ 近 7 日观点统计               │
├──────────────────────────────┤
│ 筛选 Tabs + 状态筛选          │
├──────────────────────────────┤
│ 观点列表（可滚动）            │
├──────────────────────────────┤
│ 底部 TabBar（固定）            │
│ - 交易(双向上下箭头) / 发表观点 / 分享(右上箭头) │
└──────────────────────────────┘
```

- TabBar 固定在底部，内容区需预留底部空间避免遮挡。
- 交易图标为双向上下箭头，分享图标为右上箭头。
- TabBar 高度与一级页面一致。
- 点击「发表观点」后，在个股详情内覆盖弹出发布观点内容层，并隐藏底部 TabBar；关闭/发布后回到个股详情。

## PersonalViewer（个人主页）

```
┌──────────────────────────────┐
│ 导航栏（固定）               │
│ - 返回 / 个人主页 / 分享(右上箭头) │
├──────────────────────────────┤
│ 资料区 + 指标区 + 观点列表     │
├──────────────────────────────┤
│ 分享提示 Toast               │
│ - 已複製個人主頁鏈接         │
└──────────────────────────────┘
```

- 分享链接使用脱敏短码（/u/{code}）。
- 社群分享缩略图文案：`twsvp.com` + `快来看看 {用户昵称} 在TWSVP上的观点`。

## FeedCard（观点详情）

```
┌──────────────────────────────┐
│ 导航栏（固定）               │
│ - 返回 / 观点详情 / 分享(右上箭头) │
├──────────────────────────────┤
│ 观点详情卡片（可滚动）        │
│ - 标的、方向、作者、状态      │
│ - 内容、点赞                 │
├──────────────────────────────┤
│ 留言区                        │
│ - 留言列表                    │
│ - 输入框 + 发布按钮           │
├──────────────────────────────┤
│ 分享提示 Toast               │
│ - 已複製觀點鏈接             │
└──────────────────────────────┘
```

- 社群分享缩略图文案：`twsvp.com` + `快来看看 {标的名称} 在TWSVP上的观点`。
- 留言列表每条右侧有「…」
  - 本人留言：可删除
  - 非本人留言：可选择「不要看到」仅对自己隐藏
