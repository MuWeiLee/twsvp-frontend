# Wireframe

## 全局设置
- 默认语言：繁体中文（zh-Hant）。

## Search（搜索）

```
┌──────────────────────────────┐
│ 导航栏（固定）               │
├──────────────────────────────┤
│ 搜索框 + 搜索按钮（固定）    │
├──────────────────────────────┤
│ 输入联想下拉选单（插入）     │
├──────────────────────────────┤
│ 内容区（可滚动）             │
│ - 结果 Tabs                  │
│ - 股票/观点/用户列表         │
│ - 空态/加载态                │
├──────────────────────────────┤
│ 免责声明（固定）             │
├──────────────────────────────┤
│ 底部 Tabbar（固定）          │
└──────────────────────────────┘
```

- 搜索框固定在导航栏下方。
- 输入联想下拉选单插入在搜索框与结果 Tabs 之间。
- 输入联想下拉选单最多显示 5 条，多选项时固定为 5 条高度。
- 精准匹配单一选项时，下拉选单高度仅一行。
- 内容区仅在搜索框与底部 Tabbar 之间滚动。
- 免责声明固定在 Tabbar 上方，不参与滚动。
- 页面切换进入时自动回到顶部，不保留上一页滚动位置。

## StockFeed（个股详情）

```
┌──────────────────────────────┐
│ 导航栏（固定）               │
├──────────────────────────────┤
│ 日K 行情卡片                 │
│ - 30 根 K 线               │
│ - 浅灰网格线背景             │
│ - 左右轴 + 时间轴 + 点击提示 │
├──────────────────────────────┤
│ 近 7 日观点统计               │
├──────────────────────────────┤
│ 筛选 Tabs + 状态筛选          │
├──────────────────────────────┤
│ 观点列表（可滚动）            │
├──────────────────────────────┤
│ 底部 TabBar（固定）            │
│ - 交易(双向上下箭头) / 发表观点 / 分享(右上箭头) │
└──────────────────────────────┘
```

- TabBar 固定在底部，内容区需预留底部空间避免遮挡。
- 交易图标为双向上下箭头，分享图标为右上箭头。
- TabBar 高度与一级页面一致。
- 点击「发表观点」后，在个股详情内覆盖弹出发布观点内容层，并隐藏底部 TabBar；关闭/发布后回到个股详情。

## PersonalViewer（个人主页）

```
┌──────────────────────────────┐
│ 导航栏（固定）               │
│ - 返回 / 个人主页 / 分享(右上箭头) │
├──────────────────────────────┤
│ 资料区 + 指标区 + 观点列表     │
├──────────────────────────────┤
│ 分享提示 Toast               │
│ - 已複製個人主頁鏈接         │
└──────────────────────────────┘
```

- 分享链接使用脱敏短码（/u/{code}）。
- 社群分享缩略图文案：`twsvp.com` + `快来看看 {用户昵称} 在TWSVP上的观点`。
