# Wireframe

## Search（搜索）

```
┌──────────────────────────────┐
│ 导航栏（固定）               │
├──────────────────────────────┤
│ 搜索框 + 搜索按钮（固定）    │
├──────────────────────────────┤
│ 内容区（可滚动）             │
│ - 结果 Tabs                  │
│ - 股票/观点/用户列表         │
│ - 空态/加载态                │
├──────────────────────────────┤
│ 免责声明（固定）             │
├──────────────────────────────┤
│ 底部 Tabbar（固定）          │
└──────────────────────────────┘
```

- 搜索框固定在导航栏下方。
- 内容区仅在搜索框与底部 Tabbar 之间滚动。
- 免责声明固定在 Tabbar 上方，不参与滚动。
